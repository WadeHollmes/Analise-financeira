# Anánise financeira
- O objetivo deste projeto é de analisar até 12 meses de sua vida finaceira, usando unicialmente numpy e matplotlib. 
- Vai ser analizado a entraida e saida de cada mês juntamente com a reserva de emergencia e investimentos.
- Essas informações vão ser armazenadas em arrays do numpy para facilitar os calculos da analize
